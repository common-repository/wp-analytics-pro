== Description ==

The plugn will display various statistical informations in graphical chart to admin section for wordpress basic contents, user details, buddypress details,bbpress forum topics & <a href="https://wordpress.org/plugins/voter-plugin/">Voter Plugin</a> details. For user related chart it will contain chart for New User Registration, User Delete, Login, Logout, Password etc.
Related to wordpress basic content charts for New Posts, Comments, Posts Update/Trash, Comment Trash/Spam etc.  If buddypress plugin activated then it will display buddypress charts for Buddypress Activity, New Updates, Birth Chart, Forum Post, Topic, New Group Joined, Profile Photo, Profile Update, Media Upload, Message, Friend Request, Notification, Gift etc.
Also display statistical chart for <a href="https://wordpress.org/plugins/voter-plugin/">Voter Plugin</a> which display up & down voting date wise.

<br /><br />
New "Ranking" feature added for post & custom posts after plugin version : 1.2.0. <br />
Ranking will be display in the wp-admin > Post listing pages. <br />

== Installation ==
1. Unzip and upload plugin folder to your /wp-content/plugins/ directory  OR Go to wp-admin > plugins > Add new Plugin & Upload plugin zip.
2. Go to wp-admin > Plugins(left menu) > Activate the plugin
3. See the plugin option link with plugin description on plugin activation page or directly access from wp-admin > Settings(left menu) > Wordpress Analytic
